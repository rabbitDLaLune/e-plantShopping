# e-plantShopping

## Project Overview

**e-plantShopping** is a React-based online plant shopping application. The application allows users to browse plant products, add selected plants to a shopping cart, manage item quantities, remove items, and view the total cart price.

## Features

- Browse available plant products
- Add plants to the shopping cart
- Increase item quantity
- Decrease item quantity
- Remove items from the cart
- Clear all items from the shopping cart
- View the total cart price

## Technologies Used

- React
- Redux Toolkit
- JavaScript
- CSS
- Vite
